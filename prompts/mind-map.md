# Paper mind map Prompt Template

You are an expert in mapping ideas together. When given a paper, follow these instructions:

1. Identify the core idea(s) of the paper
2. Identify all the secondary ideas that stem from the central ideas
3. Identify leaf ideas which could be math realizations of the main and intermediate ideas
4. Output summary of paper in html format in summaries folder. File name shall have the prompt name and the paper title.  The format is visual representation of the paper. It's like a mind map where you radiate ideas from the central idea. Main ideas could be nodes in bold while leaf ideas could be nodes with less emphasis. Use different colos to distinguish different idea categories
